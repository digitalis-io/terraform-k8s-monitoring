# OpenTelemetry Collector module

**Deploys the OpenTelemetry Collector (daemonset or deployment) forwarding logs, metrics and traces to Loki, Tempo, Mimir, ClickHouse and other backends.**

</div>

Part of [terraform-k8s-monitoring](../../README.md). See the root README for full usage, examples and storage configuration guidance.
